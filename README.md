# 💻 IT Support Toolkit

CLI based toolkit untuk membantu pekerjaan IT Support seperti troubleshooting jaringan, cek sistem, dan maintenance dibuat dengan Python.

---

## ✨ Features

- 🖥️ System Information (CPU, RAM, Disk)
- 🌐 Ping Test (cek koneksi internet)
- 📡 IP Information
- 🔄 Flush DNS
- 🧹 Disk Cleanup (Temp files)
- ⚙️ Service Check
- 🛠️ Auto Fix Network
- ❤️ Health Check (SFC Scan)
- 📄 Export Report
- 📝 Logging Activity

## 📦 Installation

Clone repository:
```bash
git clone https://github.com/agusadhitama/it-support-toolkit.git
cd it-support-toolkit
```
Install dependencies:
```bash
pip install -r requirements.txt
```
---

## ▶️ Usage

Jalankan program:
```bash
python main.py
```
atau:
```bash
py main.py
```
